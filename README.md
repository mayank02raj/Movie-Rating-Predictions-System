# Movie Rating Prediction System (CIS - 550 Final Project)

The final project for the CIS 550 Advance Machine Learning course related to the prediction of the box office performance of movies over the years.

## Files

- `run_scripts.sh` - Used to automate the running of scripts and to generate reports.

## Resources used

- For report generation:
  - [Pandoc Lua Filters](https://github.com/pandoc/lua-filters)
  - [Lua Filters Documentation](https://github.com/pandoc/lua-filters)
